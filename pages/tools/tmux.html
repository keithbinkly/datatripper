tmux
Terminal Multiplexer • Session Management

What is it?

tmux lets you run multiple terminal sessions inside a single window. You can split panes, create tabs (windows), and most importantly—sessions persist even if you close the terminal or disconnect from SSH.

Think of it as a window manager for your terminal.

Installation

brew install tmux

# Verify
tmux -V

Core Concepts

SessionA collection of windows. Persists in background. Named (e.g., "project-x").
WindowA tab within a session. Has a number and optional name.
PaneA split within a window. Multiple panes visible at once.
PrefixKey combo before commands. Default: Ctrl+b

Session Management

# From outside tmux (shell)
tmux                        # start new unnamed session
tmux new -s myproject       # start named session
tmux ls                     # list sessions
tmux attach -t myproject    # attach to session
tmux kill-session -t name   # kill session

# From inside tmux (prefix commands)
Ctrl+b d                    # detach from session
Ctrl+b $                    # rename session
Ctrl+b s                    # switch session (interactive)

Window Management

# All commands start with prefix (Ctrl+b)
Ctrl+b c          # create new window
Ctrl+b ,          # rename window
Ctrl+b n          # next window
Ctrl+b p          # previous window
Ctrl+b 0-9        # go to window by number
Ctrl+b w          # list windows (interactive)
Ctrl+b &          # kill window

Pane Management

# Splitting
Ctrl+b %          # split vertical (left/right)
Ctrl+b "          # split horizontal (top/bottom)

# Navigation
Ctrl+b ←↓↑→       # move between panes
Ctrl+b o          # cycle through panes
Ctrl+b q          # show pane numbers, then press number to jump
Ctrl+b z          # zoom pane (toggle fullscreen)

# Resizing
Ctrl+b Ctrl+←↓↑→  # resize pane (hold Ctrl)
Ctrl+b :resize-pane -D 10   # resize down 10 lines

# Other
Ctrl+b x          # kill pane
Ctrl+b {          # swap pane left
Ctrl+b }          # swap pane right
Ctrl+b space      # cycle layouts

Copy Mode (Scrollback)

tmux has its own scrollback buffer. You enter "copy mode" to scroll and select text:

Ctrl+b [          # enter copy mode
q                 # exit copy mode

# In copy mode (vi keys by default):
h j k l           # navigate
Ctrl+u / Ctrl+d   # page up/down
/                 # search
n / N             # next/prev match
space             # start selection
Enter             # copy selection

Ctrl+b ]          # paste

Configuration

Config file: ~/.tmux.conf

# Example .tmux.conf

# Change prefix to Ctrl+a (easier to type)
unbind C-b
set -g prefix C-a
bind C-a send-prefix

# Enable mouse
set -g mouse on

# Start windows at 1, not 0
set -g base-index 1
setw -g pane-base-index 1

# Easier splits (| and -)
bind | split-window -h
bind - split-window -v

# Vi mode for copy
setw -g mode-keys vi

# Reload config
bind r source-file ~/.tmux.conf \; display "Reloaded!"

# Reload after editing:
# tmux source-file ~/.tmux.conf

Plugin Manager (tpm)

# Install tpm
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm

# Add to .tmux.conf
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
set -g @plugin 'tmux-plugins/tmux-resurrect'   # save/restore sessions

run '~/.tmux/plugins/tpm/tpm'

# Install plugins: Ctrl+b I (capital i)

Common Workflows

# Dev setup: editor + terminal + git
tmux new -s project
Ctrl+b %              # split vertical
nvim .                # left pane: editor
Ctrl+b →              # move right
Ctrl+b "              # split right pane horizontal
# top-right: shell for commands
Ctrl+b ↓              # move to bottom-right
lazygit               # bottom-right: git

# End of day
Ctrl+b d              # detach

# Next day
tmux attach -t project    # everything still running
