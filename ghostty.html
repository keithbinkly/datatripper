Ghostty
Terminal Emulator • GPU-Accelerated

What is it?

Ghostty is a new terminal emulator created by Mitchell Hashimoto (founder of HashiCorp, creator of Vagrant, Terraform). It's GPU-accelerated, fast, and intentionally simple.

It replaces Terminal.app, iTerm2, Alacritty, Kitty, etc. The philosophy is: do one thing well—be a terminal.

Installation

brew install --cask ghostty

# Or download from ghostty.org

Why Ghostty?

SpeedGPU-accelerated rendering, very low latency
NativeUses native UI toolkit (not Electron), feels right on macOS
SimpleMinimal config, sensible defaults
ModernProper Unicode, ligatures, true color
Open sourceRecently open-sourced (was invite-only beta)

Configuration

Config file: ~/.config/ghostty/config

It's a simple key=value format. No JSON/YAML nesting.

# Example ~/.config/ghostty/config

# Font
font-family = JetBrains Mono
font-size = 14

# Theme (many built-in)
theme = Dracula
# or: theme = Catppuccin Mocha
# or: theme = Tokyo Night

# Window
window-padding-x = 10
window-padding-y = 10
window-decoration = true

# Cursor
cursor-style = block
cursor-style-blink = false

# Shell integration
shell-integration = zsh

# Keybindings (optional)
keybind = cmd+t=new_tab
keybind = cmd+w=close_surface

Built-in Themes

Ghostty ships with many themes. Some popular ones:

  • Dracula
  • Catppuccin Mocha / Latte / Frappe / Macchiato
  • Tokyo Night
  • Gruvbox Dark / Light
  • Nord
  • One Dark
  • Solarized Dark / Light
# List all available themes
ghostty +list-themes

Keybindings

Default macOS keybindings work. Some useful ones:

Cmd+TNew tab
Cmd+WClose tab
Cmd+NNew window
Cmd+DSplit right
Cmd+Shift+DSplit down
Cmd+[ / ]Previous/next split
Cmd+Shift+EnterToggle fullscreen
Cmd++/-Increase/decrease font size
Cmd+0Reset font size

Shell Integration

Ghostty can integrate with your shell for features like:

  • Cmd+click to open URLs
  • Working directory tracking per tab
  • Semantic prompt zones (know where commands start/end)
# Add to config
shell-integration = zsh   # or bash, fish

Ghostty vs Others

vs iTerm2Faster, simpler. iTerm2 has more features but heavier.
vs AlacrittyBoth fast/GPU. Ghostty more polished UI, native feel.
vs KittyBoth modern. Kitty more features, Ghostty simpler.
vs Terminal.appGhostty much faster, better Unicode/color support.

Note on tmux

Ghostty has built-in splits and tabs. You might wonder: do I need tmux?

Yes, still use tmux because:

  • Sessions persist after closing Ghostty
  • Works over SSH (Ghostty splits don't)
  • Consistent keybindings across any terminal
  • More powerful session/window management

Use Ghostty as the display layer, tmux as the session layer.
