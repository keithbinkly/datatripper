Claude Code
AI Coding Agent • Anthropic

What is it?

Claude Code is Anthropic's official CLI tool for AI-assisted coding. It runs in your terminal, reads your codebase, and can write/edit files and run commands—with your approval.

It's "agentic"—meaning Claude can take multi-step actions to accomplish tasks, not just answer questions.

Installation

# Requires Node.js
npm install -g @anthropic-ai/claude-code

# First run will prompt for authentication
claude

Basic Usage

# Navigate to your project
cd ~/code/myproject

# Start Claude Code
claude

# Now you can chat:
> explain what this project does
> find all the API endpoints
> add input validation to the user form
> write tests for the auth module

Slash Commands

/helpShow available commands
/clearClear conversation context
/compactSummarize context (save tokens)
/configView/edit configuration
/costShow token usage and cost
/doctorCheck installation health
/initCreate CLAUDE.md for project
/reviewReview recent changes

CLAUDE.md - Project Instructions

Create a CLAUDE.md file in your repo root to give Claude persistent context:

# CLAUDE.md

## Project Overview
This is a Next.js e-commerce app with Prisma ORM.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Prisma + PostgreSQL
- Tailwind CSS

## Conventions
- Use functional components
- Prefer server components when possible
- API routes in /app/api
- Database models in /prisma/schema.prisma

## Common Commands
- `npm run dev` - start dev server
- `npm run db:push` - push schema changes
- `npm test` - run tests

## Important Files
- /lib/auth.ts - authentication logic
- /lib/db.ts - database client

How Claude Code Works

Claude Code can:

  • Read files in your project (with permission)
  • Write/edit files (shows diff, you approve)
  • Run shell commands (shows command, you approve)
  • Search codebase (grep, find)
  • Chain multiple actions together

It's "human in the loop"—you see what it wants to do and approve each action.

Effective Prompting

# Be specific about what you want
> refactor the checkout flow to use React Query instead of useEffect

# Reference specific files
> look at /lib/auth.ts and add rate limiting

# Ask for explanations first
> explain how the payment integration works, then suggest improvements

# Incremental changes
> first show me the plan, then implement step by step

MCP Servers

Claude Code supports MCP (Model Context Protocol) servers for extended capabilities:

  • File system access (beyond current directory)
  • Database queries
  • API integrations
  • Custom tools
# Configure in ~/.claude/config.json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@anthropic-ai/mcp-server-filesystem", "/path/to/allowed/dir"]
    }
  }
}

Tips

  • Start sessions fresh for new tasks (/clear)
  • Use /compact when context gets long
  • Write good CLAUDE.md—it saves repeated explanation
  • Review diffs carefully before approving writes
  • Ask Claude to explain before implementing complex changes
  • Combine with lazygit: Claude writes, you review/commit

Authentication

Claude Code supports:

  • Claude Pro/Max subscription (login via browser)
  • API key (ANTHROPIC_API_KEY environment variable)
# With API key
export ANTHROPIC_API_KEY=sk-ant-...
claude

# Or login with subscription
claude
# Follow browser prompt
