The Semantic Layer as Agent Memory

I've been experimenting with MCP server integration for dbt, and something clicked this week. The semantic layer isn't just about consistent metrics—it's the vocabulary that lets AI agents understand what the business actually means. When an agent asks "what's revenue?", MetricFlow doesn't just return a number. It returns context: how it's calculated, what dimensions apply, which filters matter. That context is exactly what LLMs need to reason about data correctly.
