G
Ghostty
Terminal Emulator

GPU-accelerated terminal. Replaces Terminal.app, iTerm2. Fast, native, minimal config.

Install
brew install --cask ghostty
Config
~/.config/ghostty/config
What to Learn
  • Font configuration
  • Theme/colors
  • Keybindings
  • That's mostly it—intentionally simple

T
tmux
Terminal Multiplexer

Multiple terminals in one window. Sessions persist through disconnects. Splits, tabs, windows.

Install
brew install tmux
Commands
tmuxstart session
tmux new -s namenamed session
tmux lslist sessions
tmux attach -t namereattach
Inside tmux (Ctrl+b prefix)
Ctrl+b %split vertical
Ctrl+b "split horizontal
Ctrl+b arrowmove panes
Ctrl+b cnew window
Ctrl+b nnext window
Ctrl+b ddetach
What to Learn
  • Custom prefix key (Ctrl+a popular)
  • Config: ~/.tmux.conf
  • Plugins via tpm
  • Copy mode for scrollback

N
nvim
Text Editor

Modern Vim fork. Keyboard-driven, extensible via Lua. Steep learning curve, very fast once mastered.

Install
brew install neovim
Modes
iinsert mode
Escnormal mode
:wsave
:qquit
:wqsave & quit
Navigation (normal)
h j k l← ↓ ↑ →
w / bnext/back word
gg / Gtop/bottom
/searchfind
Editing
dddelete line
yycopy line
ppaste
u / Ctrl+rundo/redo
What to Learn (big rabbit hole)
  • Config: ~/.config/nvim/init.lua
  • Starter: LazyVim, NvChad, AstroNvim
  • Motions: ciw, di", etc.
  • Plugins: telescope, treesitter, lsp

OpenCode
AI Coding Agent (OSS)

Open source alternative. Multiple providers (Claude, GPT, Gemini, local). Free models included.

Install

Launch
cd repo && opencode

  • LSP integration
  • Multi-session support

Typical Workflow

# Start day
ghostty                          # open terminal
tmux new -s myproject            # new session
cd ~/code/myproject

# Split into panes
Ctrl+b %                         # split right
nvim .                           # editor in left pane
Ctrl+b →                         # move to right pane
lazygit                          # git in right pane

# Open AI agent in new window
Ctrl+b c                         # new tmux window
claude                           # or opencode

# End of day
Ctrl+b d                         # detach tmux (session keeps running)

# Next day
tmux attach -t myproject         # everything still there
1 tmux basics 30 min
2 lazygit 30 min
3 nvim + LazyVim hours/days
4 Ghostty config 15 min
5 Claude/OpenCode learn as you go
