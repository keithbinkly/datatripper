OpenCode
AI Coding Agent • Open Source

What is it?

OpenCode is an open-source AI coding agent, similar to Claude Code but model-agnostic. It supports 75+ LLM providers including Claude, GPT, Gemini, and local models.

Built by the SST team (creators of the serverless framework). It's free, open source, and ships with free models for zero-config start.

Installation

# Quick install
curl -fsSL https://opencode.ai/install | bash

# Or via npm
npm install -g opencode

# Or via brew
brew install sst/tap/opencode

# Launch
cd your-repo
opencode

Using Your Claude Subscription

If you have Claude Pro or Max, you can use it with OpenCode:

# Launch opencode
opencode

# Use settings/config to login with Anthropic
# Or set environment variable
export ANTHROPIC_API_KEY=sk-ant-...

Provider Configuration

Configure providers via environment variables or config:

# OpenAI
export OPENAI_API_KEY=sk-...

# Anthropic
export ANTHROPIC_API_KEY=sk-ant-...

# Google
export GOOGLE_API_KEY=...

# Just have Ollama running, OpenCode will detect it

Zen - Optimized Models

OpenCode offers "Zen"—a curated set of models specifically tested and benchmarked for coding agents:

  • Validated models that work well for coding
  • Consistent performance across providers
  • No guessing which model is best

Learn more at opencode.ai/zen

OpenCode vs Claude Code

Model supportOpenCode: 75+ providers | Claude Code: Claude only
Open sourceOpenCode: Yes | Claude Code: No
Free tierOpenCode: Yes (bundled models) | Claude Code: No
UIOpenCode: Rich TUI | Claude Code: Simple TUI
Anthropic subBoth support Claude Pro/Max login
MCPClaude Code has mature MCP support

When to Use Which?

  • Claude Code if you're all-in on Claude and want official Anthropic support
  • OpenCode if you want flexibility, open source, or to try different models
  • Both are fine—they're just CLI tools, use whichever fits the task

Themes

OpenCode has a themeable TUI. Configure in settings or config file.
